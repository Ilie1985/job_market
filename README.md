# Job Market Intelligence Pipeline

## Overview
This project collects job listing data, cleans and stores it, and analyzes demand for technical skills such as Python, SQL, Excel, and Power BI.

## Tools Used
- Python
- Requests
- BeautifulSoup
- Selenium
- Pandas
- Matplotlib
- SQLite
- Jupyter Notebook
- GitHub

## Project Structure
- `scripts/` Python scripts for scraping, cleaning, storing, and analysis
- `data/` raw and cleaned datasets
- `outputs/` generated charts
- `jobs.db` SQLite database
- `notebooks/` analysis notebook

## Workflow
1. Scrape job listings
2. Clean and standardize the data
3. Store the cleaned data in SQLite
4. Analyze skills, locations, and salary patterns
5. Visualize findings

## Example Questions Answered
- What skills appear most often?
- Which locations have the most jobs?
- Which roles mention Python, SQL, Excel, or Power BI?
- What salary ranges appear most often?
